#### ENG
A simple Snake game built with Pygame.

## Features
- Red apple: +1 score, +1 length (always available)
- Yellow apple: +3 score, +3 length (spawns every 5s, lasts 5s)
- Brown apple: -3 score, -3 length (spawns every 5s, lasts 5s)
- Pause: ESC (or P on pause screen)
- Restart: R (after game over)

## Controls
- Move: Arrow keys or WASD
- Pause: ESC
- Quit: Close window (or Q on game over)

## Requirements
- Python >= 3.8, < 3.11
- Pygame

## Install
```sh
pip install pygame
```
